CSS 분리 관리

CSS는 원활한 프로젝트의 진행 및 유지 관리를 위하여, 사용 목적에 따라 분리 하여 관리 한다.

  • @resource
    • css
      • lib
      • _base.css
      • _button.css
      • _font.css
      • _form.css
      • _global.css
      • _layout.css
      • project.css
    • images
      • @temp
      • @uio
_base.css 브라우져별 초기화
_button.css 버튼
_font.css 폰트
_form.css 폼요소 및 필드셋 간격
_global.css 공통
_layout.css 레이아웃
project.css 프로젝트 or 카테고리 단위의 CSS
